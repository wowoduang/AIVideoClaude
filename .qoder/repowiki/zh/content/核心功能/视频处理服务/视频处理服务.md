## 简介
本文件面向NarratoAI的视频处理服务，系统性梳理自动剪辑、片段拼接、时序匹配、素材管理、场景构建、时间线分配等能力。重点覆盖：
- 视频裁剪：时间轴定位、精确切割、质量保持与硬件加速兼容性
- 视频合并：多片段排序、原声/解说混合、格式统一与音频对齐
- 素材管理：视频检索、下载、解析与缓存策略
- 场景构建：基于字幕/关键帧的智能分镜
- 时间线分配：预算估算、文本截断与节奏控制
- **新增** 视觉证据融合：多模态证据整合与场景证据生成
- **新增** 帧选择系统：智能代表帧选择与稀疏采样算法
- **新增** 媒体时长探测：增强的音频时长检测与缓存机制
- **新增** 预检检查：脚本完整性验证与TTS结果检查
- 实际使用案例与性能优化建议

- 脚本驱动：VideoService接收脚本，提取时间戳，交由裁剪模块生成子片段

- 合并模块：统一分辨率、分离/混合音频、最终合成
- 素材管理：检索/下载/保存，提供稳定输入
- 场景构建：字幕/关键帧驱动的智能分镜
- 时间线分配：预算约束下的文本截断与节奏控制
- **新增** 视觉证据融合：WebUI生成脚本时的多模态证据整合
- **新增** 帧选择系统：智能代表帧选择与场景覆盖
- **新增** 媒体时长探测：增强的音频时长检测与缓存

## 详细组件分析

### 视频裁剪模块（clip_video.py）
- 时间戳解析与动态结束时间计算，支持毫秒与时长叠加
- 编码器配置与硬件加速选择，针对不同平台与GPU类型提供最优参数
- 智能错误分析与多级回退策略：滤镜链错误、硬件加速错误、编码器错误、通用回退
- 针对OST（Original Sound Track）三种模式的差异化处理：纯解说、纯原声、混合
- FFmpeg命令构建与执行，包含像素格式、音频参数、优化标志等

```mermaid
flowchart TD
Start(["开始裁剪"]) --> Parse["解析时间戳"]
Parse --> CalcEnd["计算结束时间"]
CalcEnd --> Detect["检测硬件加速"]
Detect --> ChooseEnc["选择编码器配置"]
ChooseEnc --> BuildCmd["构建FFmpeg命令"]
BuildCmd --> Exec["执行命令"]
Exec --> Validate{"输出有效?"}
Validate --> |是| Done(["完成"])
Validate --> |否| Analyze["分析错误类型"]
Analyze --> FilterErr{"滤镜链错误?"}
FilterErr --> |是| Compat["兼容性回退"]
FilterErr --> |否| HWErr{"硬件加速错误?"}
HWErr --> |是| Soft["软件编码回退"]
HWErr --> |否| EncErr{"编码器错误?"}
EncErr --> |是| Basic["基本编码回退"]
EncErr --> |否| Fallback["通用回退"]
Compat --> Exec
Soft --> Exec
Basic --> Exec
Fallback --> Exec
```

## 故障排查指南
- 裁剪失败
  - 检查时间戳格式与边界（起止时间、总时长）
  - 查看错误类型：滤镜链、硬件加速、编码器
  - 优先尝试兼容性回退与软件编码回退
- 合并失败
  - 确认视频片段存在且可读
  - 检查音频提取与混合命令
  - 使用无音频合并作为兜底
- 字幕/音轨异常
  - 校验SRT格式与编码
  - 调整字幕位置与描边参数
  - 智能音量分析失败时回退到默认音量
- 关键帧提取失败
  - 尝试超级兼容性方案（PNG→JPG）
  - 禁用硬件加速或更换编码器
- **新增** 帧选择失败
  - 检查关键帧文件名格式是否符合预期
  - 验证场景边界与关键帧时间戳的对应关系
  - 确认稀疏采样算法的输入参数合理
- **新增** 视觉证据融合失败
  - 检查LLM响应格式是否为有效的JSON
  - 验证帧路径与场景ID的映射关系
  - 确认视觉观测的解析和过滤逻辑
- **新增** 媒体时长探测失败
  - 检查ffprobe安装和可执行权限
  - 验证媒体文件的完整性和可读性
  - 查看调试日志中的具体错误信息
- **新增** 预检检查失败
  - 检查脚本数组是否为空
  - 验证必需字段的存在性和有效性
  - 确认TTS结果与脚本片段的对应关系
